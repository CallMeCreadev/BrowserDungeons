# BrowserDungeons / Platformer

## Goal
Build a simple 2D browser platformer like super metroid

## Game design
- Move left/right (A/D or Arrow Keys)
- Jump (Space/W/Up)
- Gravity + friction
- Simple collision against solid tiles/platforms
- Fall damage and if fall below worlds, death.
- Shoot projectiles (K/Shift/Ctrl)
